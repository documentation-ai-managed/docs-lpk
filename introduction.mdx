## Overview

Lpk provides a centralized space for you to organize, manage, and publish all your project documentation. You create custom pages, embed interactive components, and collaborate with your team in real-time. Whether you document APIs, user guides, or internal wikis, Lpk scales with your needs.

Start by connecting your GitHub repository or uploading Markdown files directly. Lpk renders MDX with full support for custom components like tabs, steps, and code groups, ensuring your docs look professional without extra setup.

<Callout kind="info">
  Lpk uses `#3B82F6` as the primary brand color. Apply it consistently for buttons, links, and accents to maintain visual unity.
</Callout>

## Brand Color and Styling Guidelines

Maintain consistency using Lpk's brand color `#3B82F6`. Define it in your CSS variables for easy theming.

<CodeGroup tabs="CSS,SCSS,Tailwind">
  ```css
  :root {
    --brand-color: #3B82F6;
  }

  .button {
    background-color: var(--brand-color);
    color: white;
  }
  ```
  ```scss
  $brand-color: #3B82F6;

  .button {
    background-color: $brand-color;
    color: white;
  }
  ```
  ```html
  <!-- Tailwind example -->
  <button class="bg-[#3B82F6] text-white px-4 py-2 rounded">
    Primary Button
  </button>
  ```
</CodeGroup>

<Callout kind="tip" kind="tip">
  Test your styling across themes. Use `{--brand-color: #3B82F6}` in custom CSS for overrides.
</Callout>
